# Python TKinter Interface

The whole point of this code is to use the GUI to display on a raspberry pi, output the GUI through the built in raspberry pi analogue video and then take that as input video into an aftermarket headunit, and VOILA!

You now have a nice pretty display of the data from your battery monitor, in your car, sans intrusive chinese spyware, and all is well in the universe!  
I have attached the necessary setup scripts for a systemd service and an install script to get this up and running on a raspberry pi without too much hassle, then
all you need to do is figure out a 12/5 Volt power source for the raspberry, and use a 3.5 mm jack to a RCA cable to transfer the video from raspberry to headunit. 

EASY!
